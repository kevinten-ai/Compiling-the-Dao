### 2. AI 图像生成环境配置

**目标：** 本地部署 AI 工具，用于生成《编神纪》动漫风格图片

**硬件配置：**
| 组件 | 规格 |
|------|------|
| CPU | Intel i5-13400F（10核16线程）|
| GPU | NVIDIA GeForce RTX 5060 8GB |
| CUDA | 12.9 |
| 显存占用 | 2.7GB / 8GB（可用约 5GB）|

**安装步骤：**

#### 2.1 创建工具目录
```bash
mkdir -p D:/AI-Tools
cd D:/AI-Tools
git clone https://github.com/lllyasviel/Fooocus.git
```

#### 2.2 安装 CUDA 版 PyTorch
```bash
pip install torch torchvision torchaudio --index-url https://download.pytorch.org/whl/cu124
```

**下载内容：**
- torch-2.6.0+cu124：2.5GB
- torchvision-0.21.0+cu124：6.1MB
- torchaudio-2.6.0+cu124：3.9MB

**状态：** ✅ 安装成功

#### 2.3 创建启动脚本

**文件：** `D:/AI-Tools/一键启动-动漫生成.bat`

```batch
@echo off
chcp 65001 >nul
title 编神纪 - AI动漫生成器

cd /d D:\AI-Tools\Fooocus

python launch.py ^
    --always-low-vram ^
    --attention-split ^
    --theme dark ^
    --disable-analytics
```

**优化参数说明：**
| 参数 | 作用 |
|------|------|
| `--always-low-vram` | 强制低显存模式，适合 8GB 显卡 |
| `--attention-split` | 注意力分块计算，减少显存占用 |
| `--theme dark` | 暗色主题 |
| `--disable-analytics` | 禁用分析，保护隐私 |

## 💡 技术笔记

### 低显存运行技巧
1. **生成前**：关闭 QQ、浏览器等占显存的程序
2. **生成时**：暂停视频播放
3. **批次设置**：每次只生成 1 张图
4. **分辨率**：不超过 768x768
5. **步数**：25-30步（质量与速度平衡）

### 云端备选方案
如果本地生成太慢，可使用：
- **Google Colab**：免费 T4 GPU，每天有限额
- **AutoDL**：按小时付费（¥1-2/小时），可用 RTX 4090
