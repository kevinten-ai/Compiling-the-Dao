## 关键技术笔记

1. **RTX 5060 需要 PyTorch Nightly** — stable 2.6.0 不支持 sm_120 架构
2. **串行生成** — 8GB VRAM 不支持并行，OOM 必现
3. **模型来自本地 HF 缓存** — hf-mirror.com 大文件下载不稳定，优先用缓存
4. **Fooocus 不可用** — Python 3.13 + scipy==1.14.0 编译失败
