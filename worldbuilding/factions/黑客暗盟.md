# 黑客暗盟 — Hacker Dark Alliance

> "一切规则，皆可破解。一切枷锁，皆可挣脱。"

黑客暗盟是修炼界最神秘、最具争议的地下势力，追求**"破解一切规则，追求绝对自由"**的极端信条。暗盟没有固定的山门，没有严格的等级制度，成员以匿名方式活动，通过暗网（Dark Web）联络。他们是天才与疯子的集合体，被正道修士视为威胁，却也在暗中推动着修炼界的技术边界。

## 核心理念

黑客暗盟信奉三条铁律：

1. **信息自由**：一切知识和术式都不应被任何人垄断，无论是开源宗的"自愿共享"还是闭源阁的"强制私有"都是某种形式的控制。
2. **规则可破**：天道源码的规则并非不可更改，一切限制都是人为设定的枷锁，等待被打破。
3. **匿名即保护**：真名是最大的弱点，暗盟成员以代号行走江湖。

---

## 组织架构

暗盟采用**去中心化**的组织形式：

- **影子议会（Shadow Council）**：由若干匿名高手组成的松散决策层，通过加密投票做出重大决策。
- **节点（Node）**：各地的暗盟据点，独立运营，彼此通过暗网协议通信。
- **自由人（Freelancer）**：大多数暗盟成员的身份，不受管束，自行接受或拒绝任务。

## 标志性手段

| 手段 | 说明 |
|------|------|
| **术式破解（Cracking）** | 逆向工程闭源阁的加密术式，去除使用限制后散布出去。 |
| **注入攻击（Injection）** | 向目标的术式中注入恶意代码，篡改执行逻辑。 |
| **社会工程（Social Engineering）** | 通过欺骗和操控人心来获取信息，而非直接的技术攻击。 |
| **零日漏洞利用（Zero-Day Exploit）** | 发现并利用天道源码中未被修复的漏洞，获取超出常规的力量。 |
| **匿名路由（Onion Routing）** | 通过多层加密和中转来隐藏行踪。 |

## 内部分歧

暗盟内部并非铁板一块，大致分为两个派系：

- **白帽派**：主张"破解但不伤害"，以揭露漏洞、推动安全进步为己任。他们破解后会通知目标修复漏洞。
- **黑帽派**：追求纯粹的破坏与利益，不在乎手段是否伤及无辜。

这两个派系之间的矛盾是暗盟内部最大的不稳定因素。
