# 开源宗 — Open Source Sect

> "Fork the Dao, Commit the Truth."
> ——开源宗祖训碑铭文

开源宗是修炼界最古老、最庞大的正道宗门之一，秉持**"术式共享，天下大同"**的核心理念。宗门坚信天道源码本应属于所有修士，任何术式的封锁与垄断都是对天道的亵渎。数千年来，开源宗将无数术式公之于众，推动了整个修炼界的进步。

## 宗门理念

开源宗认为：

1. **天道源码属于所有人**：任何修士都有权阅读、学习、改进术式。
2. **共享即力量**：术式在共享中不断被优化、被验证、被迭代，最终达到最强的状态。
3. **透明即安全**：公开的术式不存在隐藏的后门（Backdoor），所有人都能审查其安全性。
4. **贡献即荣耀**：向宗门贡献优秀术式的修士将获得崇高的声望（Star）。

## 核心设施

### 源码碑林（Source Stele Forest）

开源宗最重要的设施，一片由无数石碑组成的广袤碑林。每块石碑上刻有一门公开的术式，任何人都可以前来学习、复制（Fork）、并提交改进（Pull Request）。源码碑林按术式类别分区，由专门的维护者（Maintainer）长老负责审核新提交的术式。

### 祖训碑

位于源码碑林入口处的巨大石碑，上刻宗门祖训：**"Fork the Dao, Commit the Truth"**（分叉天道，提交真理）。每位弟子入门时都要在此碑前立誓。

### 未合并禁地（Unmerged Forbidden Zone）

开源宗深处的一片封禁之地，存放着历代被**拒绝合并（Rejected Merge）**的危险术式。这些术式或威力过于恐怖，或存在不可修复的致命缺陷，或违反宗门伦理准则。未合并禁地由重构境以上的长老轮流看守，严禁弟子擅入。

## 宗门架构

- **掌门/主维护者（Chief Maintainer）**：宗门最高领导者，负责最终的术式审核与宗门决策。
- **核心弟子/核心贡献者（Core Contributor）**：对宗门术式库有重大贡献的弟子，拥有直接合并术式的权限。
- **内门弟子/贡献者（Contributor）**：通过考核的正式弟子，可提交术式改进。
- **外门弟子/使用者（User）**：初入宗门的弟子，以学习和使用公开术式为主。

## 与其他势力的关系

| 势力 | 关系 |
|------|------|
| **闭源阁** | 理念对立，长期冲突。开源宗视闭源阁的术式垄断为修炼界的毒瘤。 |
| **黑客暗盟** | 复杂。开源宗欣赏其自由精神，但反对其破坏性手段。 |
| **审计圣殿** | 合作关系。开源宗的透明性与审计圣殿的安全审查理念相合。 |
| **炼数学院** | 友好。双方在数据相关术式上有大量合作。 |
| **底层硬修派** | 中立偏友好。开源宗尊重其对底层优化的执着。 |

## 宗门特色

- **Issue大会**：定期召开的宗门讨论大会，任何弟子都可以提出问题（Issue）和改进建议。
- **Code Review制度**：所有提交到源码碑林的新术式都必须经过至少两位高阶弟子的审查。
- **版本节**：每当宗门术式库迎来重大版本更新时举行的庆典。
