# 术语词典 — Term Glossary

> 本词典收录《编神纪》世界观中的核心术语，建立小说概念与计算机科学（Computer Science）概念之间的映射关系。

本书的世界深度融合了修炼体系与计算机科学概念。以下术语表将帮助读者理解每个小说名词背后对应的CS概念及其在故事中的具体含义。

---

## 基础概念

| 小说术语 | CS概念 | 英文 | 释义 |
|---------|--------|------|------|
| 天道源码 | 世界底层代码 | Heavenly Dao Source Code | 构成世界一切规则的根本代码，由初代编译者所写。修炼的终极目标便是理解并改写天道源码。 |
| 初代编译者 | 造物主/初始程序员 | The Ancient Creator | 远古时代编写天道源码、创造世界的至高存在。其真实身份与目的是最大的谜团。 |
| 灵力/算力 | 计算资源 | Computing Power | 修士的力量来源，驱动一切术式运行的基础资源。如同计算机的CPU算力。 |
| 丹田 | 操作系统内核 | Kernel (Dantian) | 修士体内存储与调度算力的核心器官。管理资源分配，是修炼的根基。 |
| 经脉 | 数据总线 | Bus (Meridians) | 连接丹田与身体各部分的灵力传输通道。决定算力传输效率。 |
| 神识 | 线程 | Thread (Consciousness) | 修士的精神意识，可并发执行多项任务。神识越强，可同时操控的术式越多。 |
| 术式 | 程序/函数 | Technique (Program/Function) | 修士施展的各类法术，本质上是对天道源码的调用与执行。 |
| 法宝 | 硬件外设 | Magic Treasure (Hardware Peripheral) | 辅助修炼与战斗的器物，本质上是特化的硬件设备。 |
| 灵石 | 电池/算力单位 | Spirit Stone (Power Unit) | 储存算力的结晶体，修炼界的通用货币和能量来源。 |

## 修炼相关

| 小说术语 | CS概念 | 英文 | 释义 |
|---------|--------|------|------|
| 突破 | 版本升级 | Breakthrough (Version Upgrade) | 修为从一个境界提升到下一个境界的过程，如同软件的大版本更新。 |
| 走火入魔 | 系统崩溃 | Qi Deviation (System Crash) | 修炼出错导致的严重后果，丹田（内核）崩溃或神识（线程）混乱。 |
| 栈溢出 | 栈溢出 | Stack Overflow | 强行运行超出自身算力承受范围的术式，导致经脉崩溃。 |
| 死循环 | 无限循环 | Infinite Loop | 神识陷入某种术式的无限执行中无法自拔，如不打断将导致神识消散。 |
| 内存泄漏 | 内存泄漏 | Memory Leak | 长期使用术式未正确释放算力，导致修为缓慢流失的慢性病。 |
| 蓝屏 | 内核崩溃 | Blue Screen (Kernel Panic) | 丹田严重崩溃的极端状态，修士陷入昏迷，需外力重启。 |
| 断点感知 | 断点调试 | Breakpoint Perception | 调试境修士的标志能力，可在术式执行的关键节点设置"断点"来分析。 |
| 多线程操控 | 多线程编程 | Multi-thread Control | 架构境修士可同时操控多条神识线程执行不同术式。 |

## 战斗相关

| 小说术语 | CS概念 | 英文 | 释义 |
|---------|--------|------|------|
| 快排剑法 | 快速排序 | Quick Sort Sword | 以分治法为核心的剑术，选定基准点后将攻击分列两侧逐层击破。 |
| 冒泡掌 | 冒泡排序 | Bubble Sort Palm | 掌力层层推进，每击将最大威胁"冒"至外层击破。稳定可靠的入门斗技。 |
| 二分追踪 | 二分搜索 | Binary Search Tracking | 每次将搜索范围缩小一半，极速锁定目标。 |
| 深度优先探索 | 深度优先搜索 | DFS Exploration | 沿一条路径极限深入直到触底后回溯。用于探索和破阵。 |
| 递归禁术 | 递归调用 | Recursion Forbidden Art | 以递归为核心的禁忌术式，威力强大但存在栈溢出的致命风险。 |
| 多线程分身术 | 多线程 | Multi-thread Clone | 分裂出多个可独立行动的分身。 |
| 分布式合击阵 | 分布式系统 | Distributed Formation | 多名修士将算力整合为一体的协同阵法。 |
| 防火墙结界 | 防火墙 | Firewall Barrier | 规则化的防御屏障，自动过滤拦截不符合规则的攻击。 |
| 注入攻击 | 代码注入 | Code Injection Attack | 将恶意代码注入对手术式中篡改其逻辑。黑客暗盟标志性手段。 |
| 死锁封印 | 死锁 | Deadlock Seal | 使多个目标互相等待资源释放，导致永久停滞。 |
| git revert | 版本回退 | Git Revert (Time Reversal) | 终极术式，将时间"回退"到之前的版本。 |
| fork | 分叉 | Fork (World Split) | 终极术式，将世界分叉为两个平行版本。 |
