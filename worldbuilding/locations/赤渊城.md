# 赤渊城 — Stack City

> "赤渊城，建在远古服务器遗迹之上，一柱擎天，上下两分。"

赤渊城是本书开篇的主要场景，一座建在远古服务器遗迹上的巨型城市。城市的核心是一根贯穿天地的**算力晶柱（Computing Power Crystal Pillar）**，整座城围绕这根晶柱分为上下两部分，恰如计算机中的**栈（Stack）**结构——后进先出，层层叠加。

## 城市结构

### 算力晶柱

一根巨大的晶体柱从地底深处贯穿至高空云端，是赤渊城的核心。晶柱本身是远古服务器遗迹的一部分，持续散发着算力波动，是整座城市算力供给的来源。晶柱按高度分为不同的算力层级，越高处算力越充沛。

### 上城区（栈顶 / Stack Top）

- **位置**：晶柱上半部分及周围高台。
- **居民**：富裕修士、宗门分支、商会、高阶修炼者。
- **特点**：算力充沛、环境优美、设施完善。修炼资源丰富。
- **代表场所**：
  - **万法交易所**：赤渊城最大的术式与资源交易市场。
  - **各大宗门驻城办事处**：开源宗、闭源阁等势力在此设有分部。
  - **审计圣殿赤渊分殿**：沈无漏驻扎之地。

### 下城区（栈底 / Stack Bottom）

- **位置**：晶柱下半部分及地下区域。
- **居民**：底层劳工、无修为平民、灰色势力。
- **特点**：算力稀薄、环境恶劣、治安混乱。生存艰难。
- **代表场所**：
  - **万象标注坊**：叶辰最初工作的地方，大量底层劳工在此从事数据标注工作。
  - **二手术式市场**：凌幽的店铺所在地，充斥着来路不明的廉价术式。
  - **废弃数据层**：下城区最深处，堆积着被遗弃的数据碎片，危险与机遇并存。

## 栈结构隐喻

赤渊城的"上下两城"结构完美映射了计算机中**栈（Stack）**的特性：

| 栈特性 | 城市映射 |
|--------|---------|
| **后进先出（LIFO）** | 新来的富裕者占据上城，原住民被压至下城 |
| **栈顶操作** | 上城区是一切活动的中心，资源优先分配给栈顶 |
| **栈底沉积** | 下城区积累着被遗忘的底层数据和人口 |
| **栈溢出（Stack Overflow）** | 当城市人口超出承载能力时发生的社会危机 |
| **栈帧（Stack Frame）** | 城市的每一层代表一个栈帧，记录着不同阶层的生活 |
