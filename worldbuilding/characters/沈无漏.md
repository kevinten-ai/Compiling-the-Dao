# 沈无漏 — Shen Wulou

> "天道之下，无人可逃审查。"

沈无漏是审计圣殿的外派执法修士，驻扎在赤渊城一带。他冷面教条，严格执行审计圣殿的每一条规则，不带任何个人感情。在他眼中，天道的法则就是绝对的正义，任何偏差都是必须修正的Bug。

## 性格特征

- **冷面**：几乎没有表情变化，说话语气如同机器读取日志——精准、冰冷、没有温度。
- **教条**：审计圣殿的规则就是他的信仰。他不会质疑规则的合理性，只会质疑执行是否到位。
- **一丝不苟**：执法过程中不接受任何通融。不论对方身份高低、理由多充分，违规就是违规。
- **内心并非无情**：在极少数情况下会流露出对"规则与人情之间矛盾"的微妙不安，但很快会被他的教条信仰压下去。

## 能力特点

作为调试境六阶的审计修士，沈无漏拥有强大的侦查与封锁能力：

| 能力 | 说明 |
|------|------|
| **代码审查之眼** | 可透视他人术式的内部结构，发现隐藏的缺陷和违规操作 |
| **日志追踪** | 回溯目标近期的行为记录 |
| **权限封印** | 临时封锁嫌疑人的部分修炼能力 |
| **断言检测（Assertion Check）** | 在对话中判断对方是否说谎 |

## 角色弧光（Character Arc）

沈无漏代表着"绝对秩序"的理念。他的内心冲突在于：当天道的规则本身存在问题时，作为规则的执行者，他应该忠于规则还是忠于正义？这个问题将随着故事发展不断拷问他。
