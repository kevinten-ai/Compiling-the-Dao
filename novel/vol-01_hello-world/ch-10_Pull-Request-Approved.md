# 第十章 Pull Request Approved

开源宗入门考核的最终结果，在次日清晨公布于初始化广场的灵码碑上。

二十三名考生，最终通过十四人。

苏沐橙排名第一。总评S，三关全优，编译考官备注："同期最强，建议直升内门候选。"

叶辰排名第十二。

倒数第三。

他的成绩单是一份充满矛盾的文档：

```
> 第一关（初始化测试）：待定（系统无法解析）
> 第二关（术法实战）：通过（资格赛胜，正式赛胜）
> 第三关（协作任务）：A-（特殊标注：适配层方案）
>
> 综合评定：通过
> 评语：能力存疑，建议观察。
> 编号：外门弟子 #2471
```

"能力存疑，建议观察。"叶辰反复看了两遍这八个字。

"翻译成代码就是：`// TODO: investigate later`。"NULL说，"不是认可，也不是否定。是一个被挂起的待办事项。"

"至少PR被approve了。"

"附带了七条review comment。但确实——merged。"

叶辰嘴角微微翘起。

他正式成为了开源宗外门弟子。编号2471。

---

入门仪式很简短。

青岚长老亲自为十四名新弟子发放了外门令牌——一枚六角形的铜质令牌，正面刻着开源宗的宗徽，背面刻着弟子编号。令牌本身就是一个微型术式载体，内置了身份认证、宗门通讯和基础权限管理功能。

"从今天起，你们是开源宗的外门弟子。"青岚长老的声音平和而郑重，"开源宗的立宗之本——'术式共享，天下大同'。你们在这里学到的一切，都可以自由传授给他人。但同时，你们也有义务将自己的修炼心得反馈给宗门——这就是开源的真义。"

他的目光缓缓扫过十四人。

"源码碑林向你们全面开放。碑林中收录了历代弟子留下的术式与功法，涵盖从码农境到重构境的完整修炼路径。你们可以自由学习、复制、改进这些术式——唯一的规则是：如果你改进了某个术式，必须将改进版提交回碑林，供后来者参考。"

"这就是开源宗的commit规范。"

仪式结束后，新弟子们被领到外门弟子的住处——山腰处一片整齐的石屋群落。每人分到一间独立的石屋，面积不大但干净整洁，比叶辰在赤渊城的废弃机箱强了十倍不止。

叶辰推开石屋的门。

屋内陈设简单：一张石床、一张石桌、一盏灵能灯、一块嵌在墙上的小型数据碑——那是每个弟子的"个人终端"，可以查看宗门公告、访问碑林索引、记录修炼日志。

他在石床上坐下。

"新的运行环境。"NULL的声音响起，带着一种叶辰已经熟悉的、不温不火的冷淡，"准备好接受更多编译错误了吗？"

叶辰没有回答，只是笑了笑。

他闭上眼睛，感受着开源宗灵气浓郁的环境在体内引起的温暖酥麻。垃圾回收术式仍在后台运行——十八年的累积还没有完全回收殆尽，但已经进入了减速阶段。NULL预测再过两三天就会"见底"。

在那之后，他的高速成长期就结束了。

他需要在这之前，尽可能多地利用开源宗的资源弥补基础。

叶辰拿起个人终端上的碑林索引，开始浏览。

---

下午。

叶辰沿着山间小路前往宗门的公共区域——外门弟子的日常修炼、领取任务、兑换资源都在这里进行。

小路在一棵巨大的古树下分叉。

叶辰走的是右边那条，通往碑林方向。他的心里还记着那块"不在编目系统中的古碑"——NULL说过，找到那块碑可能是解决注释代码共振问题的关键。

但他还没走出几步，就停住了。

路边的石阶上，坐着一个人。

一个邋遢的、散发着酒气的中年男人。

他靠着路旁的石柱，一条腿伸直，一条腿曲起，手里抱着一个油腻腻的酒葫芦。修炼服皱巴巴的，上面还有几处酒渍。头发半长不短地披散着，遮住了大半张脸。

从外表看，他就像是宗门里负责打扫卫生的杂役。

但NULL忽然安静了。

这种安静不是NULL惯常的"等待数据"式沉默，而是一种——警惕。

"怎么了？"叶辰在心中问。

NULL过了整整两秒才回答。

"我无法读取他的算力等级。"

叶辰的脚步微微一顿。

"读不到？是因为太低还是——"

"不是太低。是被屏蔽了。他的算力波动被某种极其高明的手法完全压缩，外泄信号约等于零。这种程度的自我屏蔽，至少需要——"NULL顿了一下，"重构境。"

重构境。

开源宗外门长老青岚才是重构境三阶。这个看起来像酒鬼的男人——

"别盯着看。"NULL说，"假装路过。"

叶辰收回目光，假装若无其事地继续走路。

他从那个男人面前走过。

酒气扑面而来——劣质灵酒特有的辛辣味道。

"小子。"

叶辰的脚步一滞。

那人的声音沙哑而慵懒，像是刚从午睡中被吵醒。

"新来的？"

"……是。"叶辰转过身。

男人从蓬乱的头发缝隙中露出一只眼睛。那只眼睛浑浊、散漫、毫无焦距——看起来喝了不少。

"外门弟子？"

"是。"

"码农境几级？"

"三阶。"

"三阶……"男人含糊地重复了一遍，咂了咂嘴，"三阶还在到处溜达，不去碑林补课？"

"正要去。"

"那去吧去吧。"男人有气无力地挥了挥手，低头又灌了一口酒。

叶辰看了他一眼，转身继续走。

"哦对了——"

男人的声音又从背后传来。

叶辰停下。

"路上掉了个东西。"

叶辰低头看——脚边的石阶缝隙里，夹着一本薄薄的小册子。

那册子不大，巴掌宽，十几页厚，封面磨损严重，上面没有标题，只在右下角用极小的字写着一行编号：

`K-API-07 // deprecated`

叶辰弯腰捡起来。

他下意识翻开了第一页。

然后他的瞳孔猛地收缩。

册子里画满了图。

不是普通的图——而是一幅接一幅精密的接口调用图。每一幅图都描绘了一组函数调用关系，箭头纵横交错，标注着参数类型、返回值、权限等级。

叶辰在标注坊工作了三年，见过无数术式的底层架构图。但这本小册子里的图——复杂程度远超他见过的任何东西。

因为这些接口图描述的不是码农境的术式，不是调试境的功法，甚至不是重构境的框架。

这些图描述的是——

"内核层接口。"NULL的声音骤然尖锐，"Kernel API。系统调用级别的接口文档。"

叶辰的手开始发抖。

内核层——那是编译境以上才有资格接触的领域。内核层接口是天道源码的外层API，通过它可以调用天道提供的系统级函数，干预局部自然法则。

这种东西出现在一个码农境三阶的外门弟子手中——

"不合理。"NULL的声音压到了极低，"这份文档的内容我能验证——它是真的。Kernel API第七版的接口规范，对应内核境修士的调用权限。但这个版本已经被标记为deprecated——废弃。和灵码碑上那个被关闭的旧接口一样。"

叶辰猛地回头。

石阶上空无一人。

那个邋遢的酒鬼男人已经消失了。连酒渍的痕迹都没有留下，仿佛从未出现过。

只有一阵淡淡的酒气还残留在空气中，被山风缓缓吹散。

叶辰握着那本薄册子，站在路中间，心脏狂跳。

"NULL，他是谁？"

"不知道。但他知道你。"

"你怎么确定？"

"一个能完美屏蔽算力波动的重构境修士，'恰好'坐在你前往碑林的必经之路上，'恰好'丢下一本废弃版本的内核层接口文档——这不是巧合。这是设计好的投放。"

叶辰低头看着手中的册子。

`K-API-07 // deprecated`

第七版。被废弃的第七版。

就像他体内被注释掉的远古代码。

就像灵码碑上被关闭的旧接口。

所有"被废弃"的东西，都指向同一个方向——某个远古时代被有意抹去的历史。

"内核层的接口文档，不该出现在码农境修士手中。"NULL的声音恢复了平静，但叶辰听得出其中压抑着的紧张，"这本册子的持有者——要么是想帮你，要么是想利用你。无论哪种，你都被某个你看不见的棋手标记了。"

叶辰将小册子贴身收好。

他抬头望向山路尽头——碑林的方向。

开源宗的源码碑林在午后的阳光中泛着温润的灰蓝色光芒。无数石碑高低错落，如同一片凝固的代码之海。

而在那片海的最深处，一块不在编目系统中的古碑正在等着他。

叶辰迈开脚步。

身后，远处的山崖上，一个抱着酒葫芦的身影靠在松树上，远远地看着那个少年走进碑林。

他的眼睛不再浑浊。

清澈得像两面镜子，映出碑林中无数石碑的倒影。

"码农境三阶，体内有远古注释代码，灵码碑上出现了旧编码格式的符文响应……"他低声自语，声音不再沙哑，而是清朗低沉，带着一种历经沧桑后的从容。

他低头看了看酒葫芦。

葫芦底部，有一行极小的字迹——与叶辰内核中注释代码的编码格式，一模一样。

"有意思。"

他仰头灌了最后一口酒，起身消失在了松林深处。
