# 第七章 git init

开源宗的入门考核在源峰山门内侧的"初始化广场"举行。

广场中央立着一块三丈高、一丈宽的巨大黑色石碑——灵码碑。碑面光滑如镜，表面流动着极淡的灰蓝色光芒。据说这块碑与天道源码的底层有微弱的连接，能够读取修炼者内核中的基础数据，生成一份标准化的评估报告。

开源宗每月初一举行一次入门考核。

这一期的考核者共有二十三人。

其中二十二人是各地推荐的年轻修士，身穿整洁的修炼服，携带各自宗门或家族的举荐信。他们之中最低也是码农境三阶，大部分是五阶以上。

第二十三人穿着一身洗得发白的粗布短衫，背着一个打了三个补丁的旧包裹，脸上还有几天前被篝火烟熏出的灰印。

叶辰缩在队伍最后面，努力降低自己的存在感。

"你的策略是'最小化进程'？"NULL在脑海中问。

"经验之谈。在标注坊，存在感越低的人越不容易被工头盯上。"

"这里不是标注坊。但策略可以保留——在你通过考核之前，暴露越少越好。"

广场正前方，三把太师椅上坐着三位考官。

居中的是一位白发苍苍的老者，面容和蔼，穿着一身洗得发白的灰色道袍——但袍上以浅银色丝线绣着极其精密的术式纹路，那是重构境修士特有的标志。他的眉目间透着一股书卷气，目光温和，偶尔扫过考生队伍时带着一种慈祥的审视。

青岚长老。重构境三阶。开源宗外门长老，负责新弟子选拔。

左侧是一位中年女修，表情严肃，不苟言笑——据苏沐橙说，这是负责术式考核的"编译考官"，专门检测考生的术式编写能力。

右侧是一位年轻男修，约莫二十出头，面如冠玉，一袭白色修炼服衬得他整个人如同一块精心雕琢的玉器。他没有坐在太师椅上，而是懒懒地倚在椅背上，手中把玩着一枚银色令牌——那枚令牌上刻着"内门"二字。

内门弟子。

"那个人。"NULL忽然说。

叶辰微微抬眼。

"码农境九阶。算力波动极其精炼——没有一丝冗余。他的经脉运行效率超出同境界平均值至少40%。"

码农境九阶——距离调试境只有一步之遥。在外门弟子中，这几乎是天花板级别的存在。

"他叫赵空明。"排在叶辰前面的一个考生小声地和同伴嘀咕，"开源宗内门排名第三，核心长老赵衡之子。快排剑法号称同辈第一，据说连调试境初期的师兄都不敢跟他对招——"

"肃静！"

青岚长老的声音不大，但算力加持之下，每一个字都清晰地传入了每个人的耳膜。广场上瞬间安静下来。

"入门考核，开始。"

青岚长老起身，走到灵码碑前，枯瘦的手掌按在碑面上。碑面上的灰蓝色光芒骤然亮起，化为一个巨大的半透明数据面板悬浮在半空。

"考核共三关。第一关——初始化测试。"

他转身面向考生，声音平和但不容置疑："每位考生依次将手掌按在灵码碑上，灵码碑会自动读取你的内核数据，生成初始化报告。报告将评估以下维度——"

面板上浮现出五个指标：

```
> 内核稳定度
> 经脉通畅度
> 算力等级
> 神识强度
> 天赋潜力
```

"五项指标，每项分为S、A、B、C、D、F六个等级。总评C级以上者通过初始化测试，进入下一关。总评D级者进入'待定区'，由考官人工复核决定去留。总评F级者——淘汰。"

简单。直接。残酷。

叶辰的心沉了一下。

他的综合评级是D-。

这意味着他大概率会落入"待定区"，能否通过取决于考官的主观判断。

"不要提前焦虑。"NULL说，"灵码碑的读取机制与苏沐橙的扫描术不同——它连接的是天道源码的底层接口，读取精度远高于个人术式。这既是风险，也是变量。"

"什么变量？"

"未知。等结果出来再分析。"

考核按照报名顺序进行。第一个上前的是一名来自赤渊城修炼世家的少年，码农境五阶，手掌按上灵码碑后，碑面迅速浮现出一份报告——

```
> 内核稳定度：B
> 经脉通畅度：B
> 算力等级：B
> 神识强度：C
> 天赋潜力：B
> 【总评：B】
```

"通过。"青岚长老点头。

人群中响起一阵低声议论——全B是不错的成绩，说明此人各方面均衡发展，没有明显短板。

接下来的考生们依次上前。

大部分人的报告在B到C之间，偶尔有一两个拿到A的——每出现一个A，人群都会发出一阵惊叹。

第十四号。

苏沐橙大步走向灵码碑。

她没有任何犹豫，右手利落地按上碑面。灰蓝色光芒猛然炸亮——比之前任何一个考生都亮了三倍不止。灵码碑甚至微微震颤了一下，仿佛被注入了超出预期的能量。

报告几乎是瞬间生成的：

```
> 内核稳定度：A
> 经脉通畅度：S
> 算力等级：A
> 神识强度：A
> 天赋潜力：S
> 【总评：S】
```

全场沉默了一瞬。

然后爆发出一阵压抑不住的惊叹。

S级。

开源宗的入门考核中，S级天赋的出现概率大约是百中取一。更何况苏沐橙在经脉通畅度和天赋潜力两项上都达到了S——这意味着她天生就拥有近乎完美的经脉结构和极高的成长上限。

"果然。"那个名叫赵空明的内门弟子终于动了一下，嘴角微微上翘，目光中闪过一丝欣赏——同时还有别的什么东西，一闪即逝，被完美的微笑遮盖。

青岚长老的眉毛挑了一下。

"苏沐橙，外门弟子推荐入考，码农境七阶。"他看了一眼苏沐橙，目光中多了几分温和，"通过。初始化评级——S。非常好。"

苏沐橙咧嘴一笑，朝叶辰的方向比了个加油的手势。

叶辰微微点头回应。

考核继续。

第十五号到第二十二号，没有特别出彩的成绩。

然后——

"第二十三号。"青岚长老低头看了一眼名册，"叶辰。"

叶辰深吸一口气，从队尾走了出来。

他能感觉到二十多双目光同时聚焦在他身上。那些目光中有好奇、有不屑、有漠然——他穿着一身打满补丁的粗布衣衫，和周围精心打扮的修炼世家子弟相比，格格不入得像一段混进正规代码库的野生脚本。

"NULL，你的封锁还稳定吗？"他在心中问。

"稳定。注释代码和旁路通道的波动已被完全屏蔽。灵码碑读取到的将是你表层数据——一个标准的码农境三阶修士。"

"表层数据……够用吗？"

"不知道。灵码碑的读取深度未知。如果它只读表层——没问题。如果它试图读取更深层的内核结构——"

NULL没说完。

叶辰已经站在了灵码碑前。

近距离看，这块黑色石碑比远处看更加压迫。碑面上流动的灰蓝色光芒不是简单的灵光，而是无数极其细微的符文在高速运转——那是天道源码的外层接口代码。

叶辰伸出右手，按上了碑面。

冰冷。

不是物理意义上的冰冷——而是一种数据层面的"寒意"。灵码碑的读取程序瞬间入侵了他的内核，像是一台扫描仪的探针刺入了芯片。

灰蓝色光芒在碑面上翻涌——

然后，停滞了。

叶辰感觉到灵码碑的读取程序在他内核的表面反复扫描，像是一个搜索引擎在遍历一份它无法索引的文档。NULL的封锁层如同一面滴水不漏的防火墙，将注释代码、旁路通道、所有非标准数据都隔绝在了扫描范围之外。

灵码碑的读取程序只能看到一个码农境三阶修士的标准表层数据。

但问题就出在这个"标准表层"上。

叶辰的表层数据本身就不标准。

他的内核是被注释代码强行激活的，没有标准的初始化流程；他的main函数缺失，灵力运行的入口是注释代码的残留模块临时充当的；他的经脉系统中有大量已回收的垃圾数据痕迹，虽然算力已经归入内核，但回收过程留下的"标记"还在。

这些东西不需要读取深层数据就能发现——它们就写在表层。

灵码碑的光芒闪烁了三次。

通常来说，报告在一息之内就能生成。

但叶辰的报告用了整整十息。

最终，碑面上浮现出了一份前所未有的报告：

```
> 内核稳定度：D
> 经脉通畅度：D
> 算力等级：C
> 神识强度：NaN
> 天赋潜力：undefined
> 【总评：无法计算】
>
> [系统备注]
> 警告：受测者内核结构与标准模板不匹配。
> 神识强度检测返回值：NaN (Not a Number)。
> 天赋潜力检测返回值：undefined (未定义)。
> 初始化报告无法生成有效总评。
> 建议：人工复核。
```

全场再次沉默。

但这一次的沉默，性质完全不同。

不是惊叹——是困惑。

NaN？undefined？

这两个词在修炼界代表着两种截然不同的异常。NaN——"不是一个有效数值"——意味着灵码碑检测到了某种数据，但无法将其转化为标准的评级结果。undefined——"未定义"——更加诡异，意味着灵码碑在对应位置根本没有找到可评估的数据。

天赋潜力返回undefined——这个人的天赋等级在天道源码的评估体系中不存在？

这要么意味着他的天赋低到了评估系统的下限之外——连F都不够格。

要么意味着别的什么。

低语声像潮水一样在考生中蔓延。

"这是什么情况……NaN？"

"从来没见过这种报告——"

"那个人谁啊？看穿着像是底层来的——"

"undefined？连天赋等级都没有？怕不是个bug——"

叶辰面无表情地收回手。

他的内心远没有表面那么平静。NULL在他的意识深处飞速运算着，分析这份报告的每一个字段。

"分析完成。"NULL的声音极低极快，"内核稳定度D、经脉通畅度D——这是你的真实表层数据，正常。算力等级C——你的算力11.3已经接近同境界平均值的75%，C级合理。"

"神识强度NaN呢？"

"你的神识被我占用了50%。灵码碑检测到的神识数据中混合了你的意识和我的代码特征——它无法区分哪部分是'人'、哪部分是'程序'，所以返回了NaN。"

"天赋潜力undefined？"

NULL沉默了一拍。

"……你的天赋潜力存储在注释代码中。被我封锁了。灵码碑试图读取，读到了一堵墙——墙后面不是'空'，也不是'低'，而是'不可访问'。所以返回了undefined。"

叶辰在心中苦笑。

NaN和undefined——一个是因为NULL的存在干扰了数据，一个是因为NULL的封锁阻挡了读取。

他最大的秘密没有被发现。

但代价是——他的报告变成了一份系统都无法解析的乱码。

"叶辰。"

青岚长老的声音从前方传来。

叶辰抬头。老人的面容依旧和蔼，但目光中多了一层审慎。他看着那份异常的报告，眉头微微蹙起。

"你的初始化报告出现了异常。灵码碑无法生成有效总评。"

"是，长老。"

"按照规矩，无法生成总评的考生，等同于未通过初始化测试。"

人群中传来几声不出意外的窃笑。

叶辰的心往下沉了一寸。

"不过——"

青岚长老话锋一转。

"开源宗的祖训中有一条：'Any code that cannot be parsed deserves a second look.' 任何无法被解析的代码，都值得再看一次。"

他转向左右两位考官："灵码碑无法解析，不代表考生本人有问题——也可能是灵码碑的检测范式不够全面。祖训允许对此类情况进行'人工复核'。二位意见如何？"

编译考官——那位表情严肃的中年女修——皱了皱眉："青岚师兄，我入宗八十年，从未见过灵码碑返回NaN和undefined。此人的内核结构恐怕极不标准，让他入宗是否存在风险——"

"所以才需要人工复核，而不是直接通过。"青岚长老平静地说，"让他参加后面两关。如果能通过，说明他的'非标准'不是缺陷；如果通不过——自然淘汰。"

编译考官沉默了一瞬，最终点了点头。

右侧，赵空明始终没有说话。他只是微微偏头，目光淡淡地扫过叶辰——像是在打量一件不怎么起眼的商品。

然后他笑了一下。

笑得很轻、很自然，像是看到了一个不值一提的小插曲。

但叶辰注意到了他的眼睛。

那双眼睛在笑容浮现的同时，并没有跟着弯起来。它们冷冷地停留在叶辰身上整整两秒——就像一个排序算法在评估一个新元素该放在队列的哪个位置。

然后移开了。

"你被他标记了。"NULL低声说。

"我知道。"

"他对你的初步评级是——'不值得关注'。这是好事。在你有足够的实力之前，对手的轻视是你最好的缓存。"

叶辰微微点头，退回了队伍中。

苏沐橙从前面挤过来，凑到他耳边低声说："没事，后面两关才是关键。第一关只是筛选，过不过都不影响最终结果——只要你后面两关表现够好，青岚长老会让你留下的。"

"你对他很了解？"

"他是我师祖辈的人物。脾气好，但有一个原则——他绝不会放弃任何一个'可能有价值'的弟子。你的报告越奇怪，他越不会轻易放你走。"

苏沐橙说完，拍了拍叶辰的肩膀，回到了自己的位置。

第一关结束。

二十三名考生中，十七人通过，四人进入待定区，两人淘汰。

叶辰是四名待定者之一。

---

考核暂休半个时辰。

考生们散开休息，三三两两地聚在一起低声交谈。叶辰独自走到广场边缘的石阶上坐下，闭上眼睛，在心中消化刚才的信息。

"NULL，灵码碑读取数据的那一瞬间，你有没有捕捉到什么？"

"有。灵码碑的读取程序在扫描你的内核时，执行了一次异常操作——它试图访问一个标准评估流程之外的地址。"

"什么地址？"

"一个被标记为'deprecated'的旧接口。这个接口在当前版本的灵码碑评估体系中已经废弃，但它的代码并未被删除——只是被跳过。而在扫描你的时候，灵码碑的程序自动调用了这个旧接口。"

"为什么？"

"我的推测是——你体内注释代码的残留特征触发了灵码碑中一段极古老的兼容性代码。那段代码识别到了一种它'认识'的数据格式，于是自动启用了对应的旧接口来尝试读取。"

叶辰的心跳加速了。

"你是说——灵码碑曾经能够读取注释代码？"

"不确定。但至少在某个远古版本中，灵码碑的评估体系包含了一套能够处理注释代码的读取协议。只不过这套协议后来被废弃了——或者说，被有意关闭了。"

有意关闭。

叶辰想到了审计圣殿——据说数百年前，他们主导了一场"注释代码清除行动"。如果灵码碑的旧接口是在那次行动中被关闭的……

"不要在这里深想。"NULL打断了他的思绪，"有人过来了。"

叶辰睁开眼。

一个修长的身影正从广场那边走过来，白色修炼服在日光下几乎发亮。

赵空明。

他走到叶辰面前，微微低头，露出一个恰到好处的礼貌微笑。

"你好，叶辰师弟。我是赵空明，内门弟子，今天是考核的辅助监督。"

他的声音温润如玉，每一个音节都打磨得恰到好处。

"刚才的报告很有意思——NaN和undefined，我在灵码碑前看过不下三百场考核，还是头一次见到。"

叶辰站起来，比赵空明矮了小半个头。

"运气不好。"他说。

"运气？"赵空明轻轻笑了，"灵码碑的读取是纯粹的数据操作，不存在运气这个变量。NaN意味着你的某项数据超出了评估体系的识别范围——这不是坏事，也不一定是好事。只是……不常见。"

他说"不常见"的时候，目光微微往下移了一寸，落在叶辰的胸口位置——内核所在的方向。

"赵师兄关注了。"叶辰不卑不亢地说。

"算不上关注。"赵空明微微摆手，笑容不变，"只是身为内门弟子，有义务留意每一位新入宗的师弟。如果后面的考核有什么困难，可以来找我——内门弟子辅导外门新人，是开源宗的传统。"

他说完，微微颔首，转身走了。

背影潇洒从容，一举一动都带着精心排列过的优雅，如同一段被反复优化过的代码——没有一个多余的操作。

"他的心率在看到你的报告时上升了11%。"NULL冷冷地说，"但在整段对话中保持恒定。这意味着他在见到你之前就已经完成了情绪调控。"

"你想说什么？"

"他不是来打招呼的。他是来采样的。"

叶辰沉默了一瞬。

"他拿到了什么？"

"你的语音频率、微表情模式、应激反应速度——对一个善于数据分析的人来说，三十秒的对话足以建立一个初步的行为模型。他现在已经知道你的大致性格特征了。"

"那他觉得我怎么样？"

"'不构成威胁'。"NULL顿了一下，"至少目前是这样。这正是我们需要的。"

叶辰收回目光，望向初始化广场中央的灵码碑。

半个时辰后，第二关即将开始——术法实战。

他需要一个术式。

一个从零编写的、逻辑自洽的、能在实战中使用的术式。

他低下头，闭上眼睛，开始在意识空间中编程。

---

广场上，所有考生都已散去。

灵码碑静静矗立在日光中，碑面上的灰蓝色光芒恢复了日常的缓慢流动。

没有人注意到——在碑面最底端、最不起眼的角落里——一行极其微弱的暗金色符文正在缓缓浮现。

那些符文的编码格式极其古老。

与碑面上现行的灰蓝色天道源码截然不同——它们更加复杂，更加精密，结构中蕴含着一种只有真正触及过天道底层的存在才能写出的优雅。

符文只持续了不到三秒，便重新隐没在碑面深处。

但在这三秒钟里，青岚长老的目光恰好扫过了灵码碑。

老人的脚步微微一顿。

他的表情没有任何变化——百年修炼磨练出的定力让他可以在任何情况下保持平静。但他的右手悄然伸入袖中，指尖在一枚随身携带的记录晶片上快速刻下了几个字。

然后他收回手，像什么都没发生过一样，转身走向后台休息室。

晶片上的字迹只有四个字：

**远古编码。**

青岚长老活了一百二十年，见过的事情比大多数人一辈子读过的书都多。

但灵码碑上出现远古编码——这是他生平第一次。

他不知道这和那个叫叶辰的少年有没有关系。

但他打算找出来。
