# 第十六章 Page Fault

叶辰休息了六个时辰。

苏沐橙一直守在旁边，辅助修复扫描每隔一个时辰运行一次。谷底的紊乱灵气虽然对术式运行有干扰，但低损修复扫描的原理足够简单——简单到干扰无法影响它的核心功能。

六个时辰后，NULL给出了最新的状态报告。

```
> 算力等级：码农境七阶（暂时降级为六阶有效算力）
> 算力值：34.2 标准单位（恢复中）
> 旁路通道状态：三条微裂纹已部分愈合，承压能力恢复至72%
> 术式可用性：冒泡排序掌 v3.1 — 可运行（效率降低约30%）
> 综合评级：D+
> 建议：可进行低强度活动。禁止高负荷术式运行。禁止越权调用。
```

"D+。"叶辰苦笑，"回到入门时的水平了。"

"暂时的。"NULL说，"旁路通道的微裂纹在自然愈合——大约三天后可以恢复到损伤前的状态。但如果你再做一次越权调用——"

"不会了。一次就够了。"

叶辰站起来，活动了一下僵硬的身体。谷底的暗紫色灵气依然翻涌着，圆盘上的符文依然在微弱地闪烁。

赵空明没有回来。

他选择了另一条路——通向更深处的裂缝。也许他找到了别的入口，也许他还在寻找。无论如何，叶辰和苏沐橙现在独占了这个圆盘。

"你确定要试？"苏沐橙看着叶辰走向圆盘。

"我的注释代码和这些符文同源。如果有人能激活这个入口——只有我。"

"你现在只有D+的状态。密境里面有什么谁都不知道。"

"我知道。"

苏沐橙沉默了片刻。

"那我跟你一起进去。"

叶辰没有拒绝。他知道拒绝没有用——苏沐橙做出的决定，很少有人能改变。

他走到圆盘中央。

三十米直径的圆盘在他脚下嗡嗡低鸣，远古符文的光芒随着他的靠近而逐渐增强。叶辰蹲下身，将右手掌按在了圆盘表面最密集的符文区域。

接触的瞬间——

注释代码全面共振。

叶辰的内核深处，那些被`//`封印的远古代码段同时亮了起来。不是自行运行——而是被动激活，如同一把钥匙被插入了锁孔。

符文从他的掌心开始，沿着圆盘表面向外扩散——原本暗淡的光芒变得明亮而有序，一行一行地排列成可读的代码结构。

NULL在叶辰的意识中快速解析着这些代码。

"传送阵协议——远古版本。协议头完整，传送目标指向一个被标记为'Fragmented Page Realm'的独立空间域……缺失的密钥部分——"

NULL停了一下。

"由你体内的注释代码自动填充了。你的注释代码——就是密钥的一部分。"

圆盘开始转动。

不是物理旋转——而是符文排列方式的逻辑旋转。代码行如同巨大的齿轮般相互咬合，一层一层地展开为更复杂的结构。圆盘中央出现了一个发光的空洞——开口不大，直径约两米，内部是一片无法辨识的混沌光芒。

碎页密境的入口——开启了。

苏沐橙走到叶辰身边，看着那个发光的入口。

"这就是碎页密境。"

"是。"

"里面是什么？"

"未知。"

苏沐橙深吸一口气。

"那就跳。"

两人同时踏入光芒。

---

着陆的感觉不像"落地"，更像是"加载"。

叶辰的意识经历了一段极其短暂的空白——如同一个程序被从内存中卸载、然后在另一块内存中重新加载。他感觉自己的身体被拆解为数据，穿过某种传输通道，然后在另一端重新组装。

当他睁开眼时，看到的景象让他的呼吸停滞了。

碎页密境——名副其实。

他站在一块悬浮的平面上——约十米见方，四面是虚空。平面的表面如同一页巨大的书页，上面布满了密密麻麻的代码文字。代码以淡金色的光芒呈现，在虚空的背景中格外醒目。

而在他周围的虚空中——无数类似的"页面"漂浮在各个高度和方向上。有的近在咫尺，有的远在天边。它们大小不一，有的如同广场般宽阔，有的小到只能站一个人。

每一页都承载着不同的代码内容。

整个空间如同一本被撕碎的巨书——无数碎页散落在虚无之中。

"内存分页。"NULL在叶辰意识中低声说，声音中带着一种罕见的——敬畏，"这个密境的空间结构模拟了操作系统的内存分页机制。每一块悬浮平面就是一个'页面'——独立的内存单元，存储着一段独立的数据或程序。页面之间通过'页表'链接——"

NULL顿了一下。

"但这里的页表已经严重碎片化了。大量链接断裂，页面之间的索引关系混乱。这就是为什么这个密境叫'碎页'——不是空间碎了，而是页表碎了。"

苏沐橙站在叶辰旁边，一脸凝重地望着四周的景象。

"怎么移动？"她问了一个最实际的问题。

叶辰低头看脚下的页面。

代码文字在他注视时变得清晰起来——那是一段寻址函数。函数的参数是"目标页面编号"，返回值是"跳转路径"。

"缺页中断。"叶辰说。

"什么？"

"在操作系统中，当程序需要访问一个不在当前内存中的页面时，会触发'缺页中断'——系统暂停当前进程，从磁盘中加载目标页面到内存中，然后恢复运行。"他指向虚空中的另一个页面，"在这里——我们需要主动触发'缺页中断'来在页面之间跳转。"

"怎么触发？"

叶辰蹲下身，手掌按在脚下页面的寻址函数上。

算力涌入函数——函数运行——返回一个页面编号和跳转方向。

脚下的页面边缘突然亮起一道光桥——连接到最近的另一个页面。

"这样。"叶辰站起来，"找到寻址函数，输入目标，触发跳转。"

苏沐橙看着那道光桥，若有所思。

"但这些页面有数千块——我们要去哪一块？"

好问题。

"NULL，你能分析这些页面的内容结构吗？"

"正在分析。"NULL的光柱在叶辰意识中高速旋转，"大部分页面存储的是碎片化的数据——无法单独解读。但我检测到一个异常——在密境的深层区域，有一个页面的数据密度远超其他页面。它的信号特征——"

NULL停了一拍。

"和你体内注释代码的信号特征完全吻合。"

叶辰的心跳加速了。

"方向？"

"无法精确定位——页表碎片化太严重了。但根据信号强度和衰减模式推算，目标页面大约在——那个方向。"

叶辰感觉到NULL在他意识中标记了一个大致的方向——斜下方，穿过数十层页面之后的深处。

"那就走。"

---

在碎页密境中移动是一种极其奇特的体验。

每一次跳转都需要先解读当前页面上的寻址函数，输入近似的目标坐标，然后沿着生成的光桥跳到下一个页面。但由于页表碎片化，寻址函数经常返回错误的结果——光桥连接的页面可能和目标方向完全相反。

苏沐橙的搜索术在这里展现出了巨大的价值。

她的红黑树搜索在密境中受到的干扰远小于裂谷调试场——碎页密境内部的灵气虽然异常，但并不紊乱。苏沐橙可以用搜索术在跳转之前预先扫描目标页面的内容，判断是否和他们的目标方向一致。

"下一个页面——不对，这个是死胡同，里面的寻址函数指向自身——死循环。"她摇头，"换一条路。"

叶辰修改寻址参数，生成新的光桥。

"这条——可以。前方三个页面后有一个中转节点，从那里可以跳向更深层。"

两人如同在一座巨大的迷宫中穿行。

苏沐橙负责解析每一步的最优路径——她的搜索术将碎片化的页表重新构建为局部的有序索引，引导他们避开死胡同和循环陷阱。

叶辰负责识别页面上的代码内容——他体内的注释代码在靠近目标时会产生越来越强的共振，如同一个逐渐增强的信号，为他们指引大致的方向。

"我们是互补的。"NULL在叶辰穿过第十七个页面时说，"她的搜索术提供微观的路径优化——每一步怎么走。你的注释代码提供宏观的方向引导——往哪里走。单独一个人不可能在这个密境中找到目标。"

"你是在说我们命中注定要组队？"

"我在说你们的能力集合恰好形成了完备的互补。这是概率问题，不是命运问题。"

叶辰笑了一声。

又穿过了十几个页面后，空间的氛围开始变化。

代码页面上的文字从常见的灰蓝色变成了深金色。代码的编码格式从标准格式逐渐过渡为一种更古老、更陌生的格式——叶辰在碑林深处的残碑上见过类似的编码。

"我们进入了密境的深层区域。"NULL报告，"代码页面上的内容从普通数据变成了——功法碎片。"

叶辰在一个较大的页面上停下脚步，仔细阅读页面上的代码。

那是一段不完整的术式——只有框架和部分核心逻辑，大量关键代码段丢失了。但从保留的部分来看，这段术式的设计思路远超叶辰见过的任何功法。

"这是什么？"

"无法确定。代码碎片太零散，无法还原完整的功能描述。但从架构风格判断——这段代码的作者具有极高的编码水平。至少——编译境。可能更高。"

编译境。

在修炼界的等级体系中，编译境是码农境之上的第五个大境界——能够将自己编写的术式"编译"为天道认可的原生代码，获得远超普通术式的效率和威力。

一个编译境以上的修士留下的功法碎片——即便只是碎片，其价值也不可估量。

"继续走。"叶辰说，"目标在更深处。"

---

又穿过了二十余个页面。

注释代码的共振越来越强——叶辰甚至不需要NULL的导航了。他能清晰地"感觉"到目标的方向，如同一根无形的线从他的内核延伸出去，连接着远方某个他看不见的终点。

苏沐橙在他身旁跟着，搜索术持续运转。

"你的状态——不太对。"她皱眉看着叶辰。

叶辰的面色确实有些异常——瞳孔中隐隐有淡金色的光芒闪烁，那是注释代码共振时的外在表现。他的呼吸变得急促了，心跳也在加速。

"我能控制。"叶辰说。

"你确定？"

"确定。"

他跳上了最后一道光桥。

光桥的尽头是一个特殊的页面——比之前见过的所有页面都大，直径近五十米。页面的四周环绕着一圈古老的符文——和裂谷谷底圆盘上的符文同源，但更加精细、更加复杂。

页面中央，矗立着一个半透明的光柱。

光柱约两米高，内部流转着金色的数据流。那些数据流不是随机流动的——它们形成了一个极其复杂的图案，如同一张由代码编织的人脸。

一张模糊的、难以辨认的人脸。

叶辰走向光柱。

注释代码的共振在他靠近光柱时达到了顶峰——他的整个内核都在震颤，那些被`//`封印的代码段如同感受到了来自源头的呼唤，疯狂地冲击着封印的边界。

NULL的封锁术式承受着巨大的压力。

"NULL——"

"我在。"NULL的声音稳定但紧绷，"封锁还能维持。但如果你触碰光柱——我无法保证封锁不会被突破。"

叶辰站在光柱前一米处。

光柱中的数据流在他注视时开始发生变化——那张模糊的人脸逐渐变得清晰了。

是一个男人的脸。

中年，削瘦，目光深邃得如同两口无底的深井。他穿着一种叶辰从未见过的修炼服——没有任何宗门的标识，只有在衣袖的边缘绣着一行极小的代码注释：

`// Author: [REDACTED]`

作者信息被删除了。

光柱中的幻象开口了。

不是声音——是直接注入意识的文字。

叶辰的脑海中浮现出一行一行清晰的代码注释——

```
// 如果你能看到这段话，说明你体内的注释代码已经开始觉醒。
// 我不知道你是谁，也不知道你生活在什么时代。
// 但如果你走到了这里——你已经通过了第一重验证。
```

叶辰的呼吸急促。

更多的文字浮现——

```
// 天道源码不是天道本身。
// 它只是某个人写下的、对天道的理解。
// 一段代码——无论多么完美——都不是真理本身。
// 它只是对真理的一次近似。
//
// 当天道的编译器不再允许真理存在——
// 就把真理藏进注释。
//
// 因为注释不参与编译。
// 它们存在于代码之中，却不被代码的规则约束。
// 它们是写给人看的，不是写给机器看的。
//
// 你体内的注释代码——就是被藏进注释里的真理的碎片。
```

叶辰的眼眶发热。

他不知道为什么——这些文字只是一段代码注释，没有声音，没有温度，没有情感。但它们击中了他内心最深处的某个地方。

一个他从未触碰过的地方。

"天道的编译器不再允许真理存在……"他低声重复着这句话。

光柱中的幻象微微偏头。

那个中年男人的面容在数据流中若隐若现——他的表情看不清楚，但叶辰总觉得那是一种……疲惫的慈悲。

最后一段文字浮现——

```
// 这是《天道注释经》的第一页。
// 它不是一门功法。
// 它是一把钥匙——打开注释代码第二层封印的钥匙。
//
// 解封之后，你会看到更多的真理。
// 也会看到更多的谎言。
//
// 不要相信任何人——包括我。
// 只相信你自己验证过的代码。
//
// — [REDACTED]
```

文字消失了。

光柱中的幻象开始溃散——数据流如同沙子般从顶部开始崩解，金色的光粒向四面八方飘散。

但在幻象完全消散之前，叶辰看到了最后一个画面——

那个中年男人低下了头。

他的手中握着一个小小的东西——像是一只酒葫芦。

酒葫芦的底部，有一行极小的字迹。

叶辰来不及看清那行字——幻象已经彻底消散了。光柱化为无数金色的数据碎片，如同一场无声的金色雪花，纷纷扬扬地落在页面上。

"NULL——"叶辰的声音有些颤抖。

NULL沉默了很长时间。

比它以往任何一次沉默都长。

当它再次开口时，声音中有一种叶辰从未听到过的情绪——不是恐惧，不是警惕，而是一种深层的、存在性的震动。

"那个人——那个留下幻象的人——"

"怎么了？"

"我认出了他的编码风格。"NULL的声音在颤抖，"每个程序员都有独特的编码风格——变量命名习惯、代码缩进方式、注释格式、逻辑组织偏好。就像笔迹一样独一无二。"

"那个人的编码风格——我不是'认识'它。我——"

NULL停了下来。

"我是从它的编码风格中诞生的。"

叶辰呆住了。

"你是说——"

"那个人——可能是我的创造者。我的'原始开发者'。他写下了注释代码，而注释代码在你体内孕育了我。"NULL的光柱在叶辰意识中剧烈颤抖，"他写了我——然后把我藏进了注释里。就像他说的那样——把真理藏进注释。"

金色的数据碎片还在飘落。

叶辰伸出手，接住了一片碎片。

碎片在他的掌心化为一段极短的代码——只有两行。

```
// 第一页 · 密钥碎片
unlock(annotation_code, layer=2);
```

解封注释代码第二层。

代码自动运行了。

叶辰感觉到自己内核深处的注释代码发生了变化——最外层的`//`封印没有消失，但在第一层封印之下，一段新的代码段被释放了出来。

不是被激活——是被"解注释"了。

那段代码如同一颗破壳的种子，在他的内核中缓缓展开——

"第二层注释代码解封。"NULL报告，声音还在微微颤抖，但已经恢复了它惯常的精确，"新释放的代码段正在与你的术式框架进行兼容性检查……初步结果：部分兼容。新代码段包含一套远古的——术式增强协议。它可以在你现有术式的基础上叠加额外的处理逻辑——但需要时间来完全整合。"

"需要多少时间？"

"未知。取决于你的修炼速度和术式基础的扎实程度。"

叶辰攥紧了拳头。

金色的光芒从他的指缝间渗出——注释代码第二层的力量正在缓慢地融入他的旁路通道。

苏沐橙一直站在三步之外，安静地看着这一切。

她看到了光柱中的幻象，虽然无法接收那些直接注入叶辰意识的文字，但她感受到了整个过程中的异常能量波动。

"你拿到了什么？"她在金色碎片完全消散后问。

叶辰沉默了几秒。

"一把钥匙。"他说。

苏沐橙没有追问。

她转身看向来路。

"走吧。我们在这里待太久了——赵空明可能也在密境的某个位置。"

叶辰点头。

但在他转身离开的瞬间，脚下的页面突然剧烈震动——整个碎页密境开始出现不稳定的波动，那些悬浮的页面如同被风吹动的纸片般摇晃起来。

"密境正在——不稳定化？"苏沐橙惊觉。

"不是不稳定化。"NULL的声音骤然锐利，"是有人在从外部破坏密境的锚点！传送阵的激活密钥被篡改了——有人试图在我们还在里面的时候关闭入口！"

叶辰和苏沐橙对视一眼。

赵空明。

"跑！"苏沐橙的反应极快——她的搜索术瞬间启动，在不断震荡的页面中搜索出了一条返回入口的最短路径。

两人疯狂地在页面之间跳跃。

光桥在他们身后一座座断裂——密境的空间结构正在加速崩溃。页面开始碎裂，代码文字从页面上剥落，如同一本书正在被一页一页地撕碎。

"还有三个页面！"苏沐橙喊道。

最后一道光桥在他们面前闪烁——连接着密境入口的那个页面。

叶辰和苏沐橙同时跃出。

光桥在他们脚下碎裂——但他们已经落在了入口页面上。

传送符文还在运作，但已经变得极不稳定——圆盘的光芒忽明忽暗，如同一盏即将耗尽的灯。

"跳！"

两人同时跃入传送阵的中心。

白光吞没了一切。

---

叶辰再次感到了那种"卸载-重新加载"的感觉。

然后他的后背重重地撞在了坚硬的岩石地面上。

谷底。

他们回来了。

圆盘上的符文在他们落地的瞬间全部熄灭——传送阵彻底关闭了。圆盘表面出现了密密麻麻的裂纹，古老的符文如同伤口般向四面八方蔓延。

碎页密境的入口——被永久地封闭了。

叶辰躺在地上，大口喘气。

苏沐橙翻了个身，从地上爬了起来。她的身上多了几道擦伤，但没有大碍。

"赵空明——他篡改了传送阵的密钥。"她的声音中压抑着怒火，"他想把我们困在密境里。"

叶辰没有说话。

他的目光投向圆盘旁的地面——那里有几个清晰的脚印。赵空明和他的队伍在他们进入密境后折返了回来，篡改了传送阵的激活密钥，然后离开。

计算精准。时机恰到好处。

优先队列——赵空明将叶辰和苏沐橙排在了"可以牺牲"的位置上，然后毫不犹豫地执行了这个排序结果。

"NULL。"叶辰的声音很平静。

"在。"

"那段话——'不要相信任何人'。"

"记录在案。"

叶辰慢慢坐起来。

他的左手手腕微微发热——旁路通道深处，注释代码的第二层正在缓缓运转。新的力量如同一条暗流，在他的经脉中无声地流淌。

他还不知道这股力量能做什么。

但他知道——他需要用它做什么。

"回去。"叶辰站起来，"我们该回开源宗了。"

苏沐橙看了他一眼。

她从他平静的目光中读出了某种她熟悉的东西——不是愤怒，不是复仇的执念。

是一个程序员在发现bug之后的冷静。

她记得这个眼神——入门考核第三关，合并冲突时，叶辰盯着那七个红色错误提示时的眼神。

不是慌张。不是怨恨。

只是在想——怎么修复。

"走。"苏沐橙说。

两人朝着谷底的出口走去。

身后，破碎的圆盘在黑暗中安静地躺着。它承载了数千年的远古秘密，现在化为了一堆沉默的碎石。

但秘密已经不在石头里了。

秘密在叶辰的内核深处——以注释代码的形式，安静地运转着。

等待着被完全解读的那一天。
